# HandTalk AI 后端服务启动指南

## 环境要求

- Python 3.11+
- Conda（推荐）或 pip
- CUDA（可选，用于 GPU 加速）

## 1. 创建 Conda 环境

```bash
# 创建新环境
conda create -n Shen python=3.11 -y

# 激活环境
conda activate Shen
```

## 2. 安装依赖

```bash
cd D:\Aprogress\Shen\Project\Back
pip install -r requirements.txt

# 安装深度学习相关依赖（用于手语识别）
pip install torch torchvision opencv-python pillow tqdm scikit-learn datasets
```

## 3. 启动后端服务

```bash
cd D:\Aprogress\Shen\Project\Back
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

或者使用热重载模式：

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## 4. 验证启动

打开浏览器访问：http://localhost:8000

或使用 curl 测试：

```bash
curl http://localhost:8000/
```

## API 接口文档

### 健康检查
- `GET /` - 根路径健康检查
- `GET /api/health` - API 健康检查

### 手语词典（已实现）
- `GET /api/v1/dictionary/search?query=关键词&search_type=hybrid&top_k=20&category=分类` - 搜索词汇
- `GET /api/v1/dictionary/list?page=1&limit=100` - 获取词汇列表
- `GET /api/v1/dictionary/{id}` - 获取单个词汇详情
- `GET /api/v1/dictionary/categories/list` - 获取所有分类

### 手语识别（需训练模型）
- `POST /api/v1/recognize/image` - 识别图片中的手语
- `POST /api/v1/recognize/video` - 识别视频中的手语
- `GET /api/v1/recognize/status` - 获取识别服务状态
- `GET /api/v1/recognize/health` - 识别服务健康检查

### 请求示例

```bash
# 搜索 "妈妈"
curl "http://localhost:8000/api/v1/dictionary/search?query=妈妈"

# 获取词汇列表（第1页，每页10条）
curl "http://localhost:8000/api/v1/dictionary/list?page=1&limit=10"

# 获取单个词汇
curl "http://localhost:8000/api/v1/dictionary/7"

# 获取分类列表
curl "http://localhost:8000/api/v1/dictionary/categories/list"

# 测试识别接口
curl "http://localhost:8000/api/v1/recognize/status"
```

## 搜索类型说明

- `keyword` - 关键词搜索（中文匹配权重高）
- `semantic` - 语义搜索（同义词扩展）
- `hybrid` - 混合搜索（默认，综合两种方式）

## 模型训练

### 训练 ResNet 迁移学习模型

```bash
cd D:\Aprogress\Shen\Project\Back

# 方式1: 使用脚本
python scripts/train_model.py

# 方式2: 使用批处理脚本
scripts\train.bat
```

### 训练说明

1. **数据集**: 使用 `ZechengLi19/CSL-News` 数据集
2. **采样**: 默认采样 5000 条数据进行快速验证
3. **模型**: ResNet18 预训练模型 + 自定义分类头
4. **训练策略**:
   - 冻结 ResNet 骨干网络
   - 只训练分类头
   - 10 个 epoch
5. **输出**: 模型保存到 `models/sign_language/`

### 训练预期

```
- 数据集大小: 5000 条样本
- 类别数量: 约 2000-3000 个唯一类别
- 训练时间: 5-15 分钟（GPU）或 20-40 分钟（CPU）
- 预期准确率: 30-50%（初始验证）
```

## 常见问题

### 1. 端口被占用

如果 8000 端口被占用，可以使用其他端口：

```bash
python -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```

### 2. 模块导入错误

确保已正确安装所有依赖：

```bash
pip install -r requirements.txt
pip install torch torchvision
```

### 3. CUDA 内存不足

如果 GPU 内存不足，可以减小 batch_size：

```python
# 在 scripts/train_model.py 中
config = ModelConfig(batch_size=16)  # 原来是 32
```

### 4. 中文显示乱码

这是 Windows 终端编码问题，不影响实际功能。

### 5. 模型未训练时识别接口返回模拟结果

这是正常行为，表示识别模块已集成但模型尚未训练。运行 `scripts/train.bat` 训练模型后即可使用真实识别功能。

## 与前端联调

前端默认连接 `http://localhost:8000/api/v1`，确保后端服务启动后再启动前端。

### 启动顺序

1. 先启动后端：`python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload`
2. 再启动前端：`cd Project/Front && npm run dev`

## 技术栈

- **后端框架**: FastAPI
- **词典搜索**: 纯内存向量检索（同义词扩展、拼音匹配、模糊搜索）
- **手语识别**: ResNet18 迁移学习（PyTorch）
- **数据存储**: 内存数据库（生产环境可替换为 PostgreSQL）
