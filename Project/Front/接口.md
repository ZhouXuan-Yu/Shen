## 📖 通用规范

### 基础信息

| 项目 | 说明 |
|------|------|
| API 基础地址 | `{API_BASE_URL}` = `http://localhost:8000/api/v1` |
| WebSocket 地址 | `{WS_URL}` = `ws://localhost:8000` |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |

### 统一响应格式

```typescript
interface ApiResponse<T> {
  code: number      // 状态码: 200 成功，其他见错误码表
  message: string   // 响应消息
  data: T          // 响应数据
}

interface PaginatedResponse<T> {
  items: T[]       // 数据列表
  total: number    // 总数量
  page: number     // 当前页码
  limit: number    // 每页数量
  totalPages: number // 总页数
}
```

### 错误码表

| code | 说明 |
|------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 🎯 实时翻译

### WebSocket 连接

**连接地址**
```
WS {WS_URL}/recognize
```

**连接流程**
```
1. 前端: new WebSocket(`${WS_URL}/recognize`)
2. 后端: 接受连接，返回 onopen
3. 前端: 发送 auth 消息（可选，带 token）
4. 后端: 返回认证结果
5. 前端: 开始发送帧数据
6. 后端: 实时返回识别结果
```

### WebSocket 消息格式

#### 客户端 → 服务端

**1. 认证消息（可选）**
```typescript
{
  type: 'auth',
  data: {
    token: string  // JWT token
  }
}
```

**2. 帧数据消息**
```typescript
{
  type: 'frame',
  data: {
    image: string,     // base64 编码的图片数据
    timestamp: number, // 时间戳（毫秒）
    width?: number,    // 图片宽度（可选）
    height?: number,   // 图片高度（可选）
  }
}
```

**3. 控制消息**
```typescript
// 开始识别
{ type: 'start' }

// 停止识别
{ type: 'stop' }

// 心跳（每 30 秒发送一次）
{ type: 'ping' }
```

#### 服务端 → 客户端

**1. 认证响应**
```typescript
{
  type: 'auth',
  data: {
    success: boolean,
    message?: string
  }
}
```

**2. 识别结果消息**
```typescript
{
  type: 'result',
  data: {
    text: string,       // 识别出的文字
    pinyin: string,     // 拼音
    meaning: string,    // 释义
    confidence: number,  // 置信度 0-100
  },
  timestamp: number     // 服务器时间戳
}
```

**3. 错误消息**
```typescript
{
  type: 'error',
  data: {
    code: number,
    message: string
  }
}
```

**4. 心跳响应**
```typescript
{ type: 'pong' }
```

### WebSocket 事件

| 事件 | 说明 |
|------|------|
| onopen | 连接成功 |
| onmessage | 收到消息 |
| onerror | 连接错误 |
| onclose | 连接关闭 |

### 使用示例

```typescript
// 前端代码参考 (composables/useWebSocket.ts)
const ws = new WebSocket(`${WS_URL}/recognize`)

ws.onopen = () => {
  console.log('连接成功')
  // 可选：发送认证信息
  ws.send(JSON.stringify({
    type: 'auth',
    data: { token: 'your-jwt-token' }
  }))
}

ws.onmessage = (event) => {
  const data = JSON.parse(event.data)
  
  if (data.type === 'result') {
    // 处理识别结果
    console.log(data.data.text)      // "你好"
    console.log(data.data.pinyin)   // "nǐ hǎo"
    console.log(data.data.confidence) // 95.5
  }
}

// 发送帧数据
function sendFrame(imageData: string) {
  ws.send(JSON.stringify({
    type: 'frame',
    data: {
      image: imageData,
      timestamp: Date.now()
    }
  }))
}
```

## 📤 上传翻译

### 接口列表

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/recognize/upload` | 上传文件进行识别 |

### POST /api/v1/recognize/upload

**请求头**
```
Content-Type: multipart/form-data
Authorization: Bearer {token}
```

**请求体 (form-data)**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | ✅ | 上传的文件（视频或图片） |
| type | string | ❌ | 识别类型: `video`/`image`，可自动识别 |
| options[topK] | number | ❌ | 返回前 K 个结果，默认 1 |

**响应成功 (200)**
```typescript
{
  code: 200,
  message: "识别成功",
  data: {
    id: "rec_abc123",           // 识别记录 ID
    results: [{
      text: "你好",
      pinyin: "nǐ hǎo",
      meaning: "用于问候别人",
      confidence: 94.5,
      startTime: 0,              // 视频开始时间（秒）
      endTime: 1.5               // 视频结束时间（秒）
    }],
    videoDuration: 3.5,          // 视频时长（秒），图片为 0
    processedFrames: 45,         // 处理的帧数
    createdAt: "2024-01-15T10:30:00Z"
  }
}
```

**使用示例**

```typescript
// 前端代码参考 (pages/translate.vue)
async function translateFile(file: File, fileType: 'video' | 'image') {
  const formData = new FormData()
  formData.append('file', file)
  
  const response = await $fetch<ApiResponse<UploadRecognitionResponse>>(
    `${API_BASE_URL}/recognize/upload`,
    {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${token}`
      },
      body: formData
    }
  )
  
  return response.data
}
```

### 识别类型枚举

| 值 | 说明 |
|---|------|
| `video` | 视频文件 (MP4, MOV, AVI) |
| `image` | 图片文件 (PNG, JPG, JPEG) |

### 支持的文件格式

| 类型 | 格式 |
|------|------|
| 视频 | MP4, MOV, AVI, MKV, WebM |
| 图片 | PNG, JPG, JPEG, GIF, WebP |

```typescript
// 前端代码参考 (stores/dictionary.ts)
const config = useRuntimeConfig()

// 获取词汇列表
async function fetchWords(page = 1, category?: string, search?: string) {
  const response = await $fetch<ApiResponse<PaginatedResponse<Word>>>(
    `${config.public.apiBase}/dictionary`,
    {
      method: 'GET',
      params: {
        page,
        limit: 20,
        category,
        search
      }
    }
  )
  return response.data
}

// 获取类别列表
async function fetchCategories() {
  const response = await $fetch<ApiResponse<WordCategory[]>>(
    `${config.public.apiBase}/dictionary/categories`,
    { method: 'GET' }
  )
  return response.data
}
```

## 📦 类型定义

### 识别相关类型

```typescript
// 实时识别结果
interface RecognitionResult {
  text: string       // 识别文字
  pinyin: string     // 拼音
  meaning: string    // 释义
  confidence: number // 置信度 0-100
  timestamp: string  // 时间戳

interface SkeletonPoints {
  hand: number[][]   // 手部关键点 [21, 3]

// 上传识别响应
interface UploadRecognitionResponse {
  id: string
```

## 📝 注意事项

1. **实时翻译**：建议使用 WebSocket 连接，可获得更低的延迟
2. **文件上传**：支持断点续传，大文件建议分片上传
3. **Token 管理**：accessToken 过期时使用 refreshToken 刷新
4. **图片格式**：建议使用 JPEG 或 PNG 格式
5. **视频格式**：建议使用 H.264 编码的 MP4 格式
