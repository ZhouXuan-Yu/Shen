# 译手 HandTalk AI - 前端项目启动指南

## 🚀 快速启动

### 1. 打开终端

**方法一：使用 VS Code 终端**
1. 打开 `Project/Front` 文件夹
2. 按 `Ctrl + `` ` 打开终端
3. 或右键点击空白处选择 "Open in Integrated Terminal"

**方法二：使用系统终端**
1. 打开 PowerShell 或 CMD
2. 运行命令：
```powershell
cd "D:\Aprogress\Shen\Project\Front"
```

---

## 📦 安装依赖

```bash
# 使用 npm
npm install

# 或使用 pnpm（推荐，速度更快）
pnpm install

# 或使用 yarn
yarn install
```

---

## 🏃 启动开发服务器

```bash
# 启动开发服务器
pnpm dev

# 或使用 npm
npm run dev
```

开发服务器运行在 **http://localhost:3000**

---

## 🔧 构建生产版本

```bash
# 构建
pnpm build

# 预览生产版本
pnpm preview
```

## 🎨 核心功能页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | 功能入口、统计数据 |
| 实时翻译 | `/recognize` | 摄像头实时识别 |
| 上传翻译 | `/translate` | 视频/图片上传 |
| 手语词典 | `/dictionary` | 词汇查询 |
| 登录 | `/auth/login` | 用户登录 |
| 注册 | `/auth/register` | 用户注册 |
| 个人中心 | `/profile` | 用户信息 |
| 历史记录 | `/profile/history` | 翻译历史 |
| 管理后台 | `/admin` | 数据分析 |

## ⚙️ 环境配置

创建 `.env` 文件：

```env
# API 基础地址
API_BASE_URL=http://localhost:8000/api/v1

# WebSocket 地址
WS_URL=ws://localhost:8000
```

## 🔗 后端 API 接口占位

```typescript
// 认证相关
POST   /api/v1/auth/register     // 用户注册
POST   /api/v1/auth/login       // 用户登录
POST   /api/v1/auth/captcha     // 发送验证码

// 识别相关
WS     /ws/recognize            // 实时识别
POST   /api/v1/recognize/upload // 文件上传识别

// 词典相关
GET    /api/v1/dictionary       // 词汇列表
GET    /api/v1/dictionary/{id}   // 词汇详情

// 用户相关
GET    /api/v1/user/profile     // 用户信息
GET    /api/v1/user/history     // 翻译历史
```

## ❓ 常见问题

**Q: 依赖安装失败？**
A: 尝试删除 `node_modules` 和 `pnpm-store` 后重新安装

**Q: 端口被占用？**
A: 修改 `nuxt.config.ts` 中的端口或运行 `pnpm dev --port 3001`

**Q: Tailwind 样式不生效？**
A: 确保配置正确后重启开发服务器
