# 简化版 ResNet 在小样本手语识别中的应用

## 项目摘要（供导师审阅）

---

## 一、研究目标

**核心问题**：在小样本（Few-shot）场景下，比较不同轻量级 CNN 模型在手语识别任务上的表现。

**研究意义**：

- 手语识别在聋哑人群交流中具有重要应用价值
- 小样本学习适用于数据稀缺的实际场景
- 为资源受限设备选择合适的模型提供参考依据

---

## 四、实验设计

### K-shot 学习设置详解

**什么是 K-shot Learning？**

K-shot Learning（K样本学习）是小样本学习的核心设置，指每个类别仅使用 K 个样本进行训练。

| K 值 | 含义 | 每类训练样本 | 总训练样本（226类） | 场景 |
|------|------|--------------|---------------------|------|
| K=1 | 极端小样本 | 1 个 | 226 个 | 数据极度稀缺 |
| K=5 | 标准小样本 | 5 个 | 1,130 个 | 典型 few-shot 设置 |
| K=10 | 中等小样本 | 10 个 | 2,260 个 | 适度数据增加 |
| K=20 | 较多样本 | 20 个 | 4,520 个 | 数据相对充足 |

**不同 K 值的研究意义**：

- K 越小 → 模型更容易过拟合 → 考验模型的泛化能力和预训练质量
- K 越大 → 模型学习更充分 → 性能通常更好但提升可能趋缓
- **不同模型对 K 值的敏感度不同**，这是本研究的核心问题之一

### 评估设置

| 设置 | 说明 |
|------|------|
| 训练数据 | 每类仅使用 K 个样本训练 |
| 评估数据 | 使用完整验证集（4,418 个样本）评估 |
| 随机种子 | 3 个种子（0/1/2）确保可复现性和统计显著性 |
| 结果报告 | 跨 3 个 seeds 报告 mean ± std |

### 训练策略

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 预训练 | ImageNet | 所有模型统一使用 |
| 优化器 | Adam | lr=0.001, weight_decay=0.0001 |
| 学习率策略 | Cosine decay | warmup 5 epochs |
| 训练轮数 | 50 epochs | 早停监控 val acc |
| 批量大小 | 8 | 适配 4GB 显存 |
| 数据增强 | 随机翻转、颜色扰动 | 轻量增强 |
| 标签平滑 | 0.1 | 防止过拟合 |

### 评估协议

| 协议 | 说明 |
|------|------|
| **Protocol A（主协议）** | 固定划分，全量验证集测试 |
| **指标** | Top-1 准确率、Top-5 准确率、Macro 准确率 |
| **统计方式** | 跨 3 个 seeds 报告 mean±std |

## 六、硬约束（不可违反）

1. **禁止修改模型结构**：只替换最后分类层，不改动任何内部结构
2. **禁止设计新模型**：只使用现成的预训练模型
3. **允许的优化范围**：
   - 超参数调整（学习率、weight decay、数据增强强度）
   - 训练策略（学习率调度、早停）
4. **可复现性要求**：固定种子，保存所有配置和结果
