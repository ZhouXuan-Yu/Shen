## 【任务1：项目技术能力识别】

### 1.1 核心技术能力清单

#### 深度学习能力

- **时序神经网络**：构建了 ResNet18 + 1D-CNN + 双向LSTM 的混合架构，用于视频序列的时序特征提取与建模
- **迁移学习**：使用预训练 ResNet18（ImageNet）进行特征提取，并设计自定义分类头进行领域适配
- **损失函数设计**：深入理解并应用 CTC Loss 解决序列对齐问题，实现无需帧级标注的端到端训练

#### 机器学习能力

## 【任务2：科研化重构】

### 2.1 科研型项目描述

**项目标题**：基于深度学习的多模态手语识别与检索系统研究

**项目摘要**：

本研究设计并实现了一个基于深度学习的多模态手语识别与检索系统，旨在解决聋哑人群与健听人群之间的无障碍沟通问题。系统采用端到端的深度学习架构，融合计算机视觉、自然语言处理和机器学习技术，实现了从手语视频/图像到中文文本的自动识别，以及基于语义相似度的文本-视频跨模态检索。

**核心研究内容**：

1. **端到端连续手语识别模型设计**：
   - 提出了一种基于 CTC（Connectionist Temporal Classification）损失函数的序列到序列识别架构
   - 模型采用 ResNet18 作为视觉特征提取器，提取每帧的 512 维特征向量
   - 设计 1D 时序卷积层进行局部时序建模，保证时间维度不变，避免 CTC 对齐错误
   - 使用双向 LSTM（2层，hidden_size=256）进行全局时序建模，捕获手语动作的前后文依赖关系
   - 通过 CTC Loss 解决视频帧序列长度（T）与输出词汇序列长度（L）不对齐的问题，实现无需帧级标注的端到端训练

2. **多模态特征提取与表示学习**：
   - 使用预训练 ResNet18（ImageNet）进行迁移学习，提取图像/视频帧的视觉特征
   - 采用 sentence-transformers 框架中的 BGE-small-zh-v1.5 模型进行中文文本的向量化表示（384维）
   - 设计视频帧采样策略（32帧/视频），平衡计算效率与识别精度
   - 实现文本预处理流程（去空白、统一标点、同义词扩展），提升检索质量

3. **跨模态检索算法研究**：
   - 构建了基于余弦相似度的文本-视频检索系统
   - 将 CE-CSL 数据集中的标注句子编码为向量，构建离线索引（embeddings.npy + metadata.json）
   - 实现查询文本的实时向量化与相似度计算，使用 np.argpartition 进行高效的 top-k 检索
   - 设计混合搜索策略，融合关键词匹配、语义搜索、同义词扩展，提升检索召回率

4. **智能系统架构与实验验证**：
   - 设计前后端分离的系统架构，将深度学习模型封装为 FastAPI 服务
   - 实现实时识别功能，支持 HTTP 轮询与 WebSocket 双模式
   - 在 CE-CSL 数据集上进行模型训练与评估，验证 CTC 模型的有效性
   - 设计并实现词典搜索、视频检索、实时识别等多个功能模块，形成完整的智能系统

**实验方法**：

- **数据集**：使用 CE-CSL（Chinese Sign Language）数据集，包含训练集、验证集、测试集
- **训练策略**：采用 Adam 优化器（学习率 1e-4），梯度裁剪（max_grad_norm=1.0），早停机制
- **评估指标**：使用 CTC Loss 作为训练损失，在验证集上评估模型性能，保存最佳检查点
- **检索评估**：通过余弦相似度计算文本-视频匹配度，使用 top-k 准确率评估检索效果

**技术创新点**：

1. **混合时序建模架构**：结合 1D-CNN 的局部特征提取与 BiLSTM 的全局时序建模，提升手语动作的识别精度
2. **端到端训练范式**：使用 CTC Loss 实现无需帧级标注的序列识别，降低标注成本
3. **跨模态检索系统**：将文本查询与视频内容映射到统一的向量空间，实现语义级别的相似度匹配
4. **多模式搜索融合**：设计关键词匹配、语义搜索、同义词扩展的混合策略，提升检索召回率

**研究意义**：

本研究在深度学习、计算机视觉、多模态数据分析等领域进行了深入探索，提出的端到端识别架构和跨模态检索方法为手语识别与无障碍沟通提供了技术支撑。系统设计体现了智能系统建模的思想，可扩展应用于智慧城市中的无障碍服务、智能交互等场景。

### 4.1 简历项目描述（5-6行版本）

**项目名称**：基于深度学习的多模态手语识别与检索系统

**技术栈**：Python, PyTorch, FastAPI, OpenCV, sentence-transformers, NumPy, Pandas, Nuxt 3, TypeScript

**项目描述**：
设计并实现了一个端到端的连续手语识别系统，采用 ResNet18 + 1D-CNN + 双向LSTM 的混合架构，使用 CTC Loss 解决序列对齐问题，实现无需帧级标注的端到端训练。构建了基于句向量模型的文本-视频跨模态检索系统，将文本查询与视频内容映射到统一向量空间，实现语义级别的相似度匹配。设计了前后端分离的智能系统架构，将深度学习模型封装为 FastAPI 服务，支持实时识别与检索推荐功能。

**个人贡献**：

- **跨模态检索系统**：构建智慧城市知识库，支持"文本查询→视频/图像/数据检索"的跨模态检索

#### 研究方向2：轻量级视觉模型在边缘计算中的应用

**研究背景**：
本项目使用 ResNet18 进行特征提取，但在实时识别场景中，模型的计算复杂度仍然较高。在智慧城市的边缘设备（如摄像头、传感器节点）中，需要轻量级、低延迟的视觉模型。类似 YOLO 的目标检测算法，手语识别也需要实时、高效的模型。

**研究内容**：
- **模型压缩与加速**：研究模型量化（INT8）、剪枝、知识蒸馏等方法，将 ResNet18 + BiLSTM 模型压缩为轻量级版本
- **轻量级架构设计**：设计 MobileNet、ShuffleNet 等轻量级 CNN 替代 ResNet18，或使用 Transformer 的轻量级变体
- **边缘计算部署**：研究模型在边缘设备（如 Jetson Nano、树莓派）上的部署与优化
- **实时推理优化**：使用 TensorRT、ONNX Runtime 等工具进行推理加速，实现低延迟的实时识别

**与刘老师研究的联系**：
- 对应"计算机视觉与目标识别"、"YOLO等目标检测算法"研究方向
- 可应用于智慧城市中的实时视频分析、边缘智能等场景

**技术路线**：
1. 对现有的 CTC 模型进行压缩实验（量化、剪枝、蒸馏）
2. 设计轻量级替代架构（MobileNet + 轻量级 LSTM）
3. 在边缘设备上部署并测试性能（延迟、准确率、功耗）
4. 与 YOLO 等目标检测算法进行对比分析，探索轻量级模型的通用设计原则

**预期成果**：
- 发表轻量级视觉模型相关论文
- 开发边缘计算部署工具链
- 申请模型压缩与加速相关专利

4. 研究模型的可解释性方法（如注意力机制、特征重要性分析）
